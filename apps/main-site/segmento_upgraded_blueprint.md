## KEY DESIGN RULES (Locked)

1. **Dark mode only** — `#0A0A0F` base. No light mode on marketing site.
2. **AppShell mockups** — every product UI shown inside persistent shell (sidebar + topbar). Never floating screenshots.
3. **Bento Grid for features** — tile size = priority. Client-side processing = largest tile always.
4. **Compliance pills everywhere** — every product card gets GDPR/HIPAA/DPDP pills. Woven in, not sectioned out.
5. **No carousels** — static or subtle animated UI preview. Never sliding image show.
6. **Max 2 fonts** — Syne (headlines) + DM Mono (data/technical). No additions.
7. **Frosted glass for redaction** — Sense mockups always show frosted overlay on PII. Visual proof of product.
8. **SLA timers in Resolve** — always visible in mockup. Proves operational understanding.
9. **Multiplayer cursors in SprintQL** — signals modern collaborative tool category instantly.
10. **DPDP badge non-negotiable** — Indian enterprise market differentiator. Always present.
